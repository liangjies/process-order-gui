# 工艺订单查询系统
一款基于 GoVCL 和 Lazarus 设计开发的订单查询桌面应用程序


### 应用结构
```shell
├── global
├── initialize
├── model
├── pkgui
├── sql
```

| 文件夹       | 说明                  | 描述                                |
| ------------ | --------------------- | ----------------------------------- |
| `global`     | 全局对象              | 全局对象                            |
| `initialize` | 初始化                | Gorm 的初始化                   |
| `model`      | 模型层                | 模型对应数据表                      |
| `pkgui`     | GUI层             | 软件界面               |
| `sql`    | service层             | 存放业务逻辑问题                    |
